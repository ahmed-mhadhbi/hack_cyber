# Hackathon Demo

Quick-start full-stack demo with React + Vite + Tailwind frontend and Node.js + Express + MongoDB backend.

## Setup

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + MongoDB
- **Logic**: Simple rule-based (no AI/ML)
